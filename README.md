# Awanta Network

Premium C2 Botnet infrastructure info website.

## Setup

1. Install Node.js (download from https://nodejs.org)
2. Run `npm install` in the project root
3. Start with `npm start`

> This repository is GitHub-ready. The `node_modules` folders are ignored and `start.bat` is optional.

## GitHub Pages hosting

This site is a static frontend and can be hosted on GitHub Pages.

1. Push the repository to GitHub.
2. Ensure the branch is named `main` or `master`.
3. GitHub Actions will deploy the repository automatically using `.github/workflows/pages.yml`.

> Note: GitHub Pages serves the static site from the root of this repository.

## Customization

- **Discord URL**: Edit `DISCORD_URL` in `app.js`
- **Content**: Modify `index.html`
- **Styling**: Modify `style.css`

## Features

- Tab-based navigation
- Particle background effects
- Animated stats counter
- Responsive design
- Terminal visual
- 50+ methods listed (Layer 4, Layer 7, Bypass)
- Pricing plans
- No backend dependencies (static only)
